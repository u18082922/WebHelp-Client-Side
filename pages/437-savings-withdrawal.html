4.3.7.1 Savings Withdrawal Requests

Description

This page allows the user to request to withdraw their savings.

Reference:

Control:

Functionality Description:

1

Select Policy

Click here to select the policy for which the savings withdrawal is for

2

Current Savings balance

The system auto-populates the current savings balance in the field

*Tip: Select the policy first in order to view the current savings balance.

3

Amount Requested input box

Click here and enter the amount desired to withdraw.

4

Submit button

Click here to finalize and submit the request

5

Got to help

This function redirects the user to the help document which explains the functions on the screen.
