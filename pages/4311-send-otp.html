4.3.1.1 Send OTP

Description

This page allows a user to send an OTP to phone number. A verification code is sent to the user’s phone number.

 

Reference:

Control:

Functionality Description:

1

Enter phone number input box

Click here to enter phone number.

Tip*: Ensure that the phone number being entered is present on person as the OTP code will be sent on this number.

2

Send OTP

Button

This button sends the OTP code to the entered phone number.

3

Cancel button

This control allows the user to cancel the operation. This takes the user back to the previous page.

•         Possible Error Messages: Phone number input box → Phone number is required
