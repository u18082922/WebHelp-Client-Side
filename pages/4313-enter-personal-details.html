4.3.1.3 Enter Personal Details

Description

This function allows a user to be able to request a quote. The user is able to enter their personal details and proceed by selecting next button which will take them to the next step which is to enter vehicle details.

Reference:

Control:

Functionality Description:

1

Personal Details Steppa

This steppa allows for the user to be able enter their personal details in the relevant input boxes.

2

Name input box

Click here to enter the client name in the input box.

3

Surname input box

Click here to enter the client surname

4

Email Address

Click here to enter the client email address. This must be an accessible email address.

5

Cell phone number

Click here to enter the cell phone number.

Tip*: Ensure that cell phone number being entered is a valid cell phone number.

6

Next button

This button allows the user to move to the next page. The system will redirect to the Vehicle details page.

7

Cancel button

This button cancels the operation. The system redirects to the home page.
